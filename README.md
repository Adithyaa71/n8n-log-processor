# Vulnerable Node.js Service 
Strictly Demo Purpose only

This repository is intentionally vulnerable. **DO NOT USE IN PRODUCTION.**

It is designed to demonstrate the code analysis capabilities of LLMs.

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## Structure

- `src/config.js`: Application configuration
- `src/index.js`: Main server entry point
- `src/auth.js`: Authentication logic
- `src/db.js`: Mock database layer
